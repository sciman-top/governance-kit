# sciman课堂工具箱 离线版使用说明

- 版本：`__VERSION__`
- 生成时间：`__GENERATED_AT__`

## 适用场景
- 无法联网或不允许安装运行时的教室电脑。
- 需要开箱即用的独立运行包。
- 需要随包附带 `.NET Desktop Runtime 10.x` 安装程序，便于离线环境分发。
- 本版与标准版分开输出，离线版目录会额外包含 `prereq/`。

## 启动方式
1. 直接运行 `sciman Classroom Toolkit.exe`。
2. 首次运行建议先打开“系统兼容性诊断”确认环境状态。
3. 若要在其他电脑先安装运行时，可使用 `prereq/` 目录中的 runtime 安装包。

## 运行要求
- Windows 10 22H2+ / Windows 11 22H2+。
- x64 架构。

## 安全提示
- 若被安全策略拦截，请优先使用离线包并按 `SHA256SUMS.txt` 校验后运行。

## 常见问题
1. 翻页/遥控不稳定  
- 确认 sciman课堂工具箱 与 PPT/WPS 权限级别一致。
- 检查安全软件是否拦截键盘钩子与输入投递。
- 在设置中优先使用消息投递模式进行兜底。

2. 被杀软拦截  
- 将程序目录加入白名单后重试。
- 使用 `SHA256SUMS.txt` 校验文件完整性，避免误判来源不明。

3. 配置保存失败  
- 配置与日志默认保存到用户目录；若策略限制用户目录写入，请联系管理员放开对应目录权限。
