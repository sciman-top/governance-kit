# sciman课堂工具箱 标准版使用说明

- 版本：`__VERSION__`
- 生成时间：`__GENERATED_AT__`

## 适用场景
- 推荐在线环境使用（允许安装运行时）。
- 包体较小，更新分发更快。

## 启动方式
1. 双击 `启动.bat`。
2. 脚本会检测 `.NET Desktop Runtime 10.x`：
- 若已安装：直接启动程序。
- 若未安装：提示先安装 `.NET Desktop Runtime 10.x`，安装完成后再启动程序。

## 运行要求
- Windows 10 22H2+ / Windows 11 22H2+。
- x64 架构。

## 安全提示
- 若被安全策略拦截，请优先使用离线包并按 `SHA256SUMS.txt` 校验后运行。

## 常见问题
1. 运行时安装失败  
- 本标准版不内置 runtime 安装包，请从微软官方下载并安装 `.NET Desktop Runtime 10.x`。
- 若学校策略禁止安装软件，请改用离线版。

2. 翻页/遥控不稳定  
- 确认 sciman课堂工具箱 与 PPT/WPS 的权限级别一致（都管理员或都非管理员）。
- 检查安全软件是否拦截全局钩子。
- 在系统诊断里查看 WPS/Office 演示窗口识别结果。

3. 程序无法保存配置  
- 避免把程序放在 `Program Files` 等受限目录。
- 建议解压到用户可写目录（如桌面、文档目录）。
